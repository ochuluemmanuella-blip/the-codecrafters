## welcome to my  sentinel string transformer
it takes in text(string), and with the following commands; ["Commands: upper, lower, cap, title, snake, reverse, exit"] changes it to the desired command.
* upper changes the whole text to uppercase, same with lower(changes text to lower case).
* cap; changes the first letter of every word to uppercase.
* Title : changes the first letter of every to uppercase except it is a connector.
* snake : changes text to snake, does not accept characters, andd replaces spaces with underscores(_).
* exit : exits the program completely.
