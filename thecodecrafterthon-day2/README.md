# Welcome to my base converter
1. type "cd thecodecrafterthon-day2" to enter the right directory
2. type "go run main.go" in the termial.
3. use the key word "convert" or "conv" followed by the value then the base in words.
* the format is as follows: <convert> <value> <base>
example: convert 1E hex
(the result is printed on new line.)
